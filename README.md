## Overview
Onboard App 2 aims to be a complete replacement of the built in Onboard Settings app, utilizing features by CSP Lua API

### Features
- Feature parity with the built app
- Refined controls (possibly an improvement)
- Multiple presets can be defined per car (e.g. roof, dash, hood cam positions)

### Usage
Place this app in your Assetto Corsa installation: `assettocorsa\apps\lua` folder
